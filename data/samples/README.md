# Sample Datasets

This directory contains sample datasets for testing and demonstration purposes.

## Files

### CSV Files
- `data.csv` - General purpose test data
- `test.csv` - Basic test dataset  
- `train.csv` - Training dataset
- `sample_submission.csv` - Sample submission format

### Healthcare Datasets
- `healthcare-dataset/` - Healthcare patient data with billing information
- `healthcare-dataset.zip` - Compressed healthcare dataset

### Kaggle Competition Data
- `ecommerce-data.zip` - E-commerce transaction data
- `health-insurance-cross-sell-prediction.zip` - Insurance cross-sell prediction dataset

## Usage

These datasets are used for:

1. **Testing DataPilot commands** - Various data formats and structures
2. **Performance benchmarking** - Large datasets for speed testing  
3. **Feature demonstrations** - Real-world examples for documentation
4. **Development** - Test edge cases and data quality issues

## Note

For comprehensive testing, additional fixtures are available in `tests/fixtures/`. This directory contains sample data that may be too large for the main fixtures directory or represents real-world datasets for demonstrations.
